[프롬프트를 잘 쓰는 팁]
  1) "무엇을" + "어디에" + "어떻게" 구체적으로 적기
     예) "index.html 의 <body> 안에" / "css/style.css 에"
  2) 한 번에 한 가지만 시키기 (작게 쪼개기)
  3) 결과가 이상하면 "이 부분이 이상해, 이렇게 고쳐줘" 라고 다시 말하기
  4) 모르는 단어가 나오면 "OOO 가 무슨 뜻이야?" 라고 물어보기
  5) 파일 이름과 폴더 이름을 정확히 적기 (예: data/top5_lists.json)


----------------------------------------------------------
STEP 1. 화면 레이아웃 만들기 (HTML)
----------------------------------------------------------

[프롬프트 예시]
"index.html 의 <body> 안에 MyTop5 사이트의 기본 화면을 만들어줘.
화면에는 다음 요소가 있어야 해:
 - 맨 위에 'MyTop5 - 영화 & 드라마 추천' 이라는 제목 (<h1>)
 - 그 아래에 검색창 (<input id='search'>, placeholder='제목/작성자/작품명 검색')
 - 그 아래에 카테고리 필터 버튼 3개 (전체 / 영화 / 드라마)
 - 그 아래에 Top5 카드들이 들어갈 영역 (<div id='list'>)
 - 오른쪽 사이드 또는 우측 상단에 '+ 새로 추가' 버튼 (<button id='add-btn'>)
아직 디자인은 신경 쓰지 말고, 구조만 잡아줘."


----------------------------------------------------------
STEP 2. 화면 디자인 (CSS)
----------------------------------------------------------

[프롬프트 예시]
"css/style.css 에 아래처럼 디자인을 적용해줘.
 - 전체 폰트는 깔끔한 sans-serif
 - 배경은 아주 옅은 회색, 카드는 흰색 + 그림자
 - 제목은 가운데 정렬, 큰 글씨
 - 검색창은 가로로 길게, 둥근 모서리
 - 카테고리 버튼은 작고 둥글게, 선택된 버튼만 색이 진하게
 - Top5 카드는 가로 폭 600px 정도, 가운데 정렬, 위아래 간격
 - '+ 새로 추가' 버튼은 화면 오른쪽 아래에 동그랗게 떠 있게"


----------------------------------------------------------
STEP 3. 예시 데이터 채우기 (JSON)
----------------------------------------------------------

[먼저 직접 할 일]
  - data/README.md 의 형식 예시를 한 번 읽어보기

[프롬프트 예시]
"data/top5_lists.json 에 예시 데이터 2개를 만들어줘.
 - 1개는 영화 카테고리, 1개는 드라마 카테고리
 - 각 묶음마다 items 가 정확히 5개
 - 형식은 data/README.md 에 적힌 형식 그대로
주석은 넣지 말고 (JSON 은 주석을 못 넣음), 순수 JSON 만 넣어줘."


----------------------------------------------------------
STEP 4. JSON 읽어서 화면에 그리기 (JS)
----------------------------------------------------------

[프롬프트 예시]
"js/main.js 에 다음 동작을 만들어줘:
 1) 페이지가 열리면 data/top5_lists.json 을 fetch 로 불러온다.
 2) 불러온 목록을 #list 안에 카드 형태로 그린다.
    카드 안에는: 제목, 작성자, 카테고리, 그리고 1~5위 항목 리스트(이름 + 코멘트).
함수는 작게 나눠서 만들어줘:
 - loadLists()        : JSON 불러오기
 - renderCards(lists) : 화면에 카드들 그리기
주석은 한국어로 자세히 달아줘."


----------------------------------------------------------
STEP 5. 검색 / 카테고리 필터 (JS)
----------------------------------------------------------

[프롬프트 예시]
"js/main.js 에 검색과 카테고리 필터를 추가해줘.
 1) #search 에 글자를 입력하면, 제목/작성자/items 의 작품 이름 중
    하나라도 그 글자를 포함하는 카드만 보이게 한다.
 2) 카테고리 버튼(전체/영화/드라마)을 누르면 그에 맞는 카드만 보인다.
 3) 검색과 카테고리 필터는 동시에 적용된다.
기존 renderCards 는 그대로 두고, 위에 필터 함수를 만들어서 호출하는 방식으로."


----------------------------------------------------------
STEP 6. '새로 추가' 폼 (JS)
----------------------------------------------------------

[프롬프트 예시]
"#add-btn 을 누르면 새 Top5 를 입력할 수 있는 폼을 보여줘.
폼에는:
 - 작성자 이름 입력
 - 카테고리 선택 (영화/드라마)
 - 묶음 제목 입력
 - 1~5위 각각 (작품 이름 + 한 줄 코멘트) 입력
'등록' 버튼을 누르면, 그 데이터를 화면 카드 목록 맨 위에 추가해줘.
일단은 새로고침하면 사라져도 괜찮아 (다음 스텝에서 저장 추가할 거야)."


----------------------------------------------------------
STEP 7. localStorage 로 저장하기 (JS, 선택)
----------------------------------------------------------

[프롬프트 예시]
"등록한 Top5 가 새로고침해도 남아 있도록 localStorage 에 저장해줘.
 - 페이지가 열릴 때: localStorage 에 저장된 게 있으면 그걸 우선 보여주고,
   없으면 data/top5_lists.json 을 보여준다.
 - 새로 등록할 때: 화면뿐 아니라 localStorage 에도 추가한다."
